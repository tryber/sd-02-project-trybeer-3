# POST localhost:3001/user/login

## Request

```
body:
{
    "email": "email@gmail.com",
    "password": "123456"
}
```

## Responses

* Success
```
{
    "status": "success",
    "token": "JSON WEBTOKEN",
    "name": "Name",
    "email": "email@gmail.com",
    "role": "client" || "admin"
}
```
* Fail
```
{
    "status": "failed",
    "code": "unauthorized",
    "message": "User not found or wrong password"
}

{
    "status": "failed",
    "code": "invalid_data",
    "message": "Missing fields"
}
```

# POST localhost:3001/user/register

## Request

```
body:
{
    "email": "email@gmail.com",
    "password": "123456",
    "name": "Seu Nome",
    "admin": true || false
}
```

## Response

* Success

```
{
    "status": "success",
    "user": {
        "name": Seu Nome",
        "password": "123456",
        "id": 1,
        "email": "email@gmail.com",
        "role": "admin" || "client"
    }
}
```

* Fail

```
{

    "message": Mensagem do erro"
}
```
